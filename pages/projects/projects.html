About us
The Website Project

A currently ongoing project, where students create a website for the club in order to help describe its activities and schedule for non-members. The website you are currently on is the project for this club, and consists of a home page, a blog, a projects page, and an about section. Currently, it has only 2 people working on it so feel free to join and help or create your own website!

The Game Design Project

An ongoing project, where students work together to create a functionable game using the Godot game engine. Reasons for using Godot is because of its easy to use design and its easy to learn scripting language, Godot script. Whether you want to create a 2d platformer with others, or make a solo project with RPG mechanics, the sky is the limit. Join this group to make your ideas into reality!

The Arcade Cabinet Project

Another ongoing project, this time with a hint of hardware. Students use both software and hardware in order to create and emulate a functionable arcade machine. This is done through downloaded software, and the usage of raspberry pies and a team working on the actual structure of the arcade cabinet. So, if you want to help these guys out with their work, and or you believe you can be of some use to the team, feel free to join!

The 3D Printing Project

Over here, you 3D print items for whatever purpose you may need it for. These 3D printed parts can be used by the Arcade Cabinet Project group, or it can just be a cool accessory you put on your body. Either way, join this group in order to learn how 3D printing works, and the variety of uses that it has!
